# PHP Project for Render Deployment

This is a simple PHP project deployed on Render.

## Features
- Basic PHP rendering
- Composer for dependency management
- `.htaccess` support (optional for Apache)

## Deployment on Render
1. Push this project to GitHub.
2. Connect the repo to Render.
3. Use the start command: `php -S 0.0.0.0:10000 -t public`
